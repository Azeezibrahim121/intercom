# 🎯 BountyBoard — Decentralized P2P Task Bounty Board

> Built on Trac Network / Intercom · Powered by the Pear Runtime

BountyBoard is a fully decentralized, peer-to-peer task bounty platform running on Trac Network.
Anyone can post tasks with TAP token rewards, claim them, and get paid — all on-chain, no middlemen.

## Features

- Post Bounties with TAP token rewards
- Claim Bounties — any peer can claim an open bounty
- Complete and Pay — reward released on completion
- Leaderboard — top earners ranked by TAP earned
- Timer Oracle — stale claims auto-reset after 30 days
- Beautiful dark-theme desktop GUI
